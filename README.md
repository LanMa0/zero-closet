# Zero-Degree Wardrobe System

A minimal, modular wardrobe design system built around the concept of **0° structure design**.

---

## 🧠 What is this?

Zero-Degree Wardrobe System is a spatial design system that rethinks traditional wardrobe structures.

It is not just furniture design — it is a **functional space system**, focused on:

- Structural minimalism (0° geometry)
- Modular storage logic
- Visual reduction (low visual noise)
- Function-driven layout design

The goal is to turn storage into a **clean, structured, and scalable system**, similar to how software systems are designed.

---

## 🎯 Core Concept

This system is based on one principle:

> Remove visual complexity. Keep only structural function.

Every design decision follows:

- No decorative elements
- No unnecessary curves or visual noise
- Pure geometric structure
- Function-first spatial organization

---

## 🧱 System Structure

The wardrobe is designed as a modular system:

- Hanging Zone (long & short clothing)
- Folding Zone (stacked storage)
- Drawer Units (small items / accessories)
- Vertical Expansion Modules
- Optional Display Module

Each module can be independently adjusted or expanded.

---

## 🎨 Design Principles

- 0° structural geometry (strict straight-line design)
- Minimal visual hierarchy
- Neutral material palette
- Hidden storage logic
- System-like spatial organization

---

## 🧠 Design Philosophy

This project treats space like software:

- Modules = components
- Zones = UI layout sections
- Movement = user interaction flow
- Storage = data structure

The wardrobe becomes a **physical operating system for clothing organization**.
